# AGOT-Local-Stability-Patch
Unofficial compatibility and stability patch for CK3 AGOT, fixing conflicts, missing localization, UI issues, and selected scripted errors in heavily modded playsets.
AGOT Local Stability Patch
AGOT Local Stability Patch is an unofficial compatibility, localization, and stability patch created for a heavily modded Crusader Kings III: A Game of Thrones playset.
Its purpose is to reduce conflicts between AGOT and selected submods, correct missing or broken localization, apply narrowly targeted safety fixes, and improve the overall stability of long campaigns without changing the intended design of the original mods.
Main Features
Compatibility fixes for selected AGOT submods.
Targeted safeguards for dragons, genes, DNA, events, CBs, Mega Wars, and other high-risk systems.
Spanish localization fixes for missing names, descriptions, traditions, innovations, tooltips, and interface elements.
Small UI and visual corrections.
Conflict-resolution files designed to load after the supported mods.
No intentional balance overhaul.
No redistribution of complete third-party mods.
Important
This patch is designed for a specific mod configuration. It may not be useful or compatible with every AGOT playset.
Place AGOT Local Stability Patch at the very bottom of the playset, after every supported mod and compatibility patch.
Recommended final load order:
-AGOT Cartographic Performance Patch
-AGOT Local Stability Patch
AGOT Local Stability Patch must be the last active mod.
Requirements
Crusader Kings III
A Game of Thrones
Any additional submods explicitly listed as dependencies on this page
Do not activate dependencies that you do not normally use merely because they are mentioned in the compatibility notes.
Save-Game Notice
A new campaign is recommended after major AGOT or submod updates.
Existing saves may retain old scripted states, characters, DNA, events, modifiers, or database information that cannot always be repaired safely by a compatibility patch.
Always keep backups of important saves.
